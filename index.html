What is Temple+?

Temple+ is an Engine Enhancement & Bugfixes mod for Temple of Elemental Evil (ToEE).

Get ToEE

If you don't have it already, you can buy ToEE from GoG Games: https://www.gog.com/game/the_temple_of_elemental_evil

Primary features:

  • Support for HD resolutions
  • Compatibility with Win10
  • Faster loading time
  • New Feats
  • New class features for levels 11+
  • New Tactical combat options
  • Gameplay Enhancements, such as flexible PC/NPC party composition
  • AI improvements
  • Pathfinding improvements
  • Many UI / convenience features, such Alt-Click to quickly sell items
  • ... and many, many BUGFIXES!
